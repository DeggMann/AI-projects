// MAIN FILE: pokemoncolosseum.py //

to start run command : python pokemoncolosseum.py

Program will ask player to Enter name
*note: player can choose length of team
*note: Ai chooses highest damage possible

// HOW TO PLAY //
- This is a turn based game which allows users to fight an opponent
- COIN flip is done to see who will go first
- pokemon are randomly selected
- certain pokemon types have an advantage over others
- a player can only use a move once
- The goal is to see which Team will have all pokemon faint first: whoever has all pokemon faint loses
- Last player standing wins, program ends

FILES:
- AIplayer.py : creates opponent 
- battle.py : calculates damage
- pokemon.py : Creates a pokemon object
- data_loader.py : parses csv files into pokemon object
- pokemoncolosseum.py [main file]: play game
